## 🎯 核心函数说明

### `checkLoginStatus(page)`
**功能**: 检测 B 站登录状态  
**实现**:
- 查找头像/用户名元素（已登录标志）
- 查找登录按钮（未登录标志）
- 返回布尔值

**选择器**:
```javascript
'.header-avatar, .user-avatar, [class*="avatar"]'
'.user-name, [class*="username"], [class*="nickname"]'
'.header-login-entry, .login-btn, a[href*="login"]'
```

---

### `uploadVideo(page, videoPath)`
**功能**: 上传视频文件  
**实现**:
1. 查找文件输入框 `input[type="file"][accept*="video"]`
2. 设置文件路径 `fileInput.setInputFiles(videoPath)`
3. 等待上传进度（最多 5 分钟）
4. 检测进度条文本包含 "100" 或 "完成"

**容错**:
- 如果找不到输入框，尝试点击上传区域
- 上传超时后提示用户手动确认

---

### `fillTitle(page, title)`
**功能**: 填写视频标题  
**实现**: 尝试多种选择器，找到第一个可用的

**选择器列表**:
```javascript
[
  'input[placeholder*="标题"]',
  'input[name*="title"]',
  'input[id*="title"]',
  '[class*="title"] input[type="text"]',
  'input[type="text"]:nth-of-type(1)',
]
```

---

### `fillDescription(page, description)`
**功能**: 填写视频描述/简介  
**选择器**:
```javascript
[
  'textarea[placeholder*="简介"]',
  'textarea[placeholder*="描述"]',
  'textarea[name*="desc"]',
]
```

---

### `addTags(page, tags)`
**功能**: 添加标签（支持多个）  
**实现**:
1. 分割标签字符串 `tags.split(',')`
2. 逐个输入并回车 `input.fill(tag); input.press('Enter')`

---

### `setCover(page, coverPath)`
**功能**: 设置封面图片  
**实现**:
1. 点击封面上传按钮
2. 找到文件输入框
3. 设置图片文件

---

### `selectPartition(page, partitionName)`
**功能**: 选择视频分区  
**实现**:
1. 查分区 ID `PARTITIONS[partitionName]`
2. 点击分区选择器
3. 选择对应分区选项

## 🛡️ 安全设计

### 1. 不自动提交
```javascript
console.log('⚠️  请手动检查以下内容:');
console.log('确认无误后，请手动点击"立即投稿"按钮');
await waitForEnter(); // 等待用户按回车
```

### 2. 不保存凭证
- 无硬编码账号密码
- 依赖浏览器 Cookie
- 每次运行需保持登录状态

### 3. 文件验证
```javascript
if (!fs.existsSync(args.video)) {
  console.error(`❌ 错误：视频文件不存在：${args.video}`);
  process.exit(1);
}
```

---

## ⚠️ 错误处理

### 1. 参数验证
- 检查必需参数（`--video`, `--title`）
- 检查文件是否存在

### 2. 页面元素查找失败
- 每个操作都有 try-catch
- 失败时提示用户手动操作
- 不中断整个流程

### 3. 超时处理
- 页面加载超时：30 秒
- 上传等待超时：5 分钟
- 超时后提示用户检查
