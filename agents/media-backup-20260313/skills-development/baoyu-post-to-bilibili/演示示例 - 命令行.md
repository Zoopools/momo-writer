## 📹 演示 1: 基础发布（必需参数）

### 场景
用户有一个视频文件，想要快速发布到 B 站。

### 命令
```bash

node bilibili-poster-v2.js \
  --video ~/Videos/my-video.mp4 \
  --title "【AI 教程】如何用 AI 工具提高效率"
```

## 🎯 使用技巧

### 1. 标题优化
```bash
# 好的标题示例：
--title "【AI 教程】01-如何用 AI 工具提高效率"
--title "挑战！用 AI 工具 1 小时完成 1 天的工作"
--title "这 5 个 AI 工具让我每天多睡 2 小时"
```

### 2. 标签选择
```bash
# 推荐 3-5 个标签，逗号分隔
--tags "AI，效率工具，生产力，教程，干货"
```

### 3. 分区选择
```bash
# 常见分区：
--partition 科技      # 科技数码类
--partition 知识      # 科普教学类
--partition 生活      # 生活记录类
--partition 游戏      # 游戏攻略类
```

### 4. 封面图建议
```bash
# 尺寸建议：1146x718 (16:9)
# 格式：JPG/PNG
# 大小：<2MB
--cover ~/Images/cover-1146x718.jpg
```

## ⚠️ 注意事项

1. **首次使用**: 需要手动登录 B 站账号
2. **实名认证**: B 站账号必须完成实名认证
3. **手动提交**: 脚本填写完成后，需手动点击"立即投稿"
4. **审核时间**: 投稿后通常 10 分钟 -2 小时审核完成
5. **视频格式**: 支持 MP4, MOV, AVI, MKV 等常见格式
6. **文件大小**: 网页端单文件最大 4GB
