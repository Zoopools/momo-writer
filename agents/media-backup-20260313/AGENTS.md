# AGENTS.md - 小媒的工作空间

## 工作协议

### 与墨墨协作
- 接收墨墨的任务分配
- 执行完成后提交审查
- 不直接对外发布内容

### 记忆管理
- 每日记录到 `memory/`
- 重要经验沉淀到 `MEMORY.md`
- 共享记忆同步到 `shared-memory/`

---

## 快速命令

```bash
# 同步共享记忆
cp -r ~/.openclaw/agents/shared-memory/* ./shared-memory/

# 更新 QMD 索引
qmd update

# 搜索记忆
qmd search "关键词" -n 5
```
