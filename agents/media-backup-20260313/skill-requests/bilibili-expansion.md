### 2. `baoyu-danmaku-design` - 弹幕互动设计
**用途**: 设计视频弹幕触发点和互动策略  
**功能需求**:
- 弹幕触发点时间轴规划
- 种子弹幕生成（10-15 条）
- 互动问题设计
- 老粉彩蛋/梗埋设
- 弹幕密度分析建议

**输出格式**:
```markdown

## 触发点
| 时间戳 | 内容时刻 | 预期弹幕反应 |
|--------|----------|--------------|
| 0:03   | 开场白   | 社区口号     |
| 2:15   | 事实揭示 | "？"震惊     |
| 5:30   | 互动问题 | 观众答案     |
```

## ⚙️ 技术实现建议

### 方案 A: API 集成
- 使用 B 站开放平台 API
- 需要开发者账号和 OAuth 认证
- 优点：稳定、官方支持
- 缺点：审核周期、功能限制

### 方案 B: Chrome CDP（推荐）
- 参考 `baoyu-post-to-wechat` 实现
- 浏览器自动化操作
- 优点：功能完整、无需 API 审核
- 缺点：依赖浏览器、需维护选择器

### 方案 C: 混合模式
- 数据读取用 API
- 发布操作用 CDP
- 平衡稳定性和功能完整性
