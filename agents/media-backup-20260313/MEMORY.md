## 🎬 B 站运营知识（2026-03-12 新增）

### 封面图设计检查清单
- [ ] 高对比度，移动端缩略图可读
- [ ] 人脸/表情角色可见（+30% CTR）
- [ ] 文字叠加：≤8 字符，粗体
- [ ] 配色匹配频道品牌识别
- [ ] 通过"滚动测试"（在 20 个缩略图中脱颖而出）

### 标题公式模板
- 【分类】好奇心钩子 + 具体细节 + 情感锚点
- 例：【硬核科普】为什么中国高铁能跑 350km/h？答案让我震惊
- 例：挑战！用 100 元在上海吃一整天，结果超出预期

### 内容规划比例
| 类型 | 比例 | 说明 |
|------|------|------|
| 支柱内容 | 40% | 深度视频，10-20 分钟，高制作价值 |
| 热点内容 | 30% | 热点响应、梗整合、及时评论 |
| 社区内容 | 20% | Q&A、粉丝互动、幕后 |
| 实验内容 | 10% | 新格式、合作、直播 |

### 关键数据目标
| 指标 | 目标值 |
|------|--------|
| 三连率（投币 + 收藏 + 点赞） | >5% |
| 弹幕密度 | >30 条/分钟（关键时刻） |
| 粉丝转化率 | 播放量→粉丝数 |
| 完播率 | 重点关注 4 分钟节点（设置模式中断） |
