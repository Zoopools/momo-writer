## 🎨 技能选择

## 📝 内容大纲

```
标题：杀戮尖塔 2 新手必知的 5 件事

1. 先了解四职业特点
   - 铁甲战士：易上手，推荐新手
   - 静默猎手：需要规划
   - 故障机器人：复杂但有潜力
   - 观者：高风险高回报

2. 卡牌不是越多越好
   - 初始卡组 5-7 张最佳
   - 每层只加 1-2 张核心卡
   - 删除废牌比加新牌重要

3. 遗物选择优先级
   - 攻击型：咖啡滤杯、红面具
   - 防御型：蓝蜡烛、冰球
   - 通用：幸运手环、药水瓶

4. 地图路线规划
   - 优先精英战（拿遗物）
   - 商店必去（买卡删牌）
   - 休息点选休息而非锻造

5. BOSS 战前准备
   - 确保有足够防御
   - 准备应对 BOSS 机制
   - 保留关键能力牌
```

## 🚀 执行步骤

1. 准备内容 markdown 文件
2. 调用 baoyu-xhs-images 技能
3. 指定风格=pixel-art, 布局=list
4. 生成信息图
5. 保存到 outbox/
