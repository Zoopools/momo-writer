## 📋 需要的 API 密钥

### 方案 A：阿里通义万象（推荐）⭐

**服务商:** DashScope（阿里）  
**模型:** wanx2.1-t2i-turbo（极速版）  
**优点:** 国内访问快、支持中文、成本低  
**价格:** 约 ¥0.02-0.05/张

**获取方式:**
1. 访问 https://dashscope.console.aliyun.com/
2. 登录阿里云账号
3. 开通"通义万相"服务
4. 创建 API Key
5. 添加到配置文件

**配置步骤:**
```bash
# 编辑小媒的 .env 文件
nano /Users/wh1ko/Documents/openclaw/agents/media/.env.local

# 添加以下内容
DASHSCOPE_API_KEY=sk-your-api-key-here

# 保存后重启 Gateway
openclaw gateway restart
```
