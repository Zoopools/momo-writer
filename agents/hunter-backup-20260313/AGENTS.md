# AGENTS.md - 小猎的工作空间

## 工作协议

### 与墨墨协作
- 接收信息搜集任务
- 执行抓取和整理
- 提交原始数据给墨墨分析

### 记忆管理
- 每日记录到 `memory/`
- 情报沉淀到 `knowledge/intelligence/`
- 共享记忆同步到 `shared-memory/`
