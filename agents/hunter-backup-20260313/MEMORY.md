## 最佳实践

### 抓取前
1. 确认目标网站 robots.txt
2. 检查是否需要登录
3. 预估数据量和频率
4. 准备备用方案

### 抓取中
1. 记录完整 URL 和时间
2. 保存原始快照
3. 处理异常和失败
4. 遵守速率限制

### 抓取后
1. 数据清洗和去重
2. 关键信息提取
3. 来源完整标注
4. 提交墨墨分析
