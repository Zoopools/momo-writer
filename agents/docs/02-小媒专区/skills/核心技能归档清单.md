## 🚀 使用方式

### 通过 OpenClaw 调用

所有技能已注册到 OpenClaw 技能系统，可通过以下方式调用：

1. **自然语言触发**:
   - "帮我把这篇文章发到公众号"
   - "生成一张小红书封面图"
   - "翻译这篇文章到中文"

2. **技能名称调用**:

   - `/baoyu-xhs-images content.md`
