🛠️ 什么是 OpenClaw 技能？

OpenClaw（外号"龙虾"）是一个开源的 AI 助手框架。

技能（Skill） 是 OpenClaw 的核心功能：

  • • ✅ 可复用的能力模块
  • • ✅ 对应固定场景或任务
  • • ✅ 一键调用，自动化执行

类比：

  • • 手机 App → 扩展手机功能
  • • OpenClaw 技能 → 扩展 AI 能力
