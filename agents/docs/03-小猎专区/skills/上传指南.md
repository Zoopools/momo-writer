## 💡 为什么不用 API 自动上传？

飞书多维表格的附件字段 API 限制较多：
- 需要额外的云空间上传权限
- 文件大小有限制（单张≤5MB）
- 需要先上传获取 file_key 再关联

**手动拖拽反而更快**，7 张图片 1 分钟搞定！
