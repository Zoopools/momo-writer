## 🎯 关键学习

### 技术学习

1. **OpenClaw 2026.3.2 限制**
   - 不支持 systemPrompt 配置
   - JSON schema 严格校验
   - SOUL.md 文件注入是正确方案

2. **技能隔离方案**
   - 物理隔离：专属目录 + 软链接
   - 逻辑隔离：SOUL.md 文件注入
   - 不修改 JSON，保持配置纯净

3. **故障恢复**
   - 备份是生命线（带时间戳）
   - 冷启动命令：`openclaw gateway start --force --allow-unconfigured`
   - 清理锁文件：`rm -f ~/.openclaw/gateway.lock`

### 架构学习

1. **混合层级架构**
   - 墨墨是总管（90% 任务）
   - 小媒是专家（10% 任务）
   - 保留灵活性，可直接找小媒

2. **权限设计原则**
   - 最小权限原则
   - 物理隔离 + 逻辑隔离
   - 不依赖单一机制

3. **版本兼容性**
   - 不修改 JSON，使用文件注入
   - 版本升级风险低
   - 配置纯净，易于恢复
