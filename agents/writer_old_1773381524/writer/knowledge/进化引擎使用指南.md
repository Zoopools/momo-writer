# 进化引擎使用指南

**版本**: 1.0.0
**创建时间**: 2026-03-12
**适用**: 所有 Agent

---

## 什么是进化引擎

进化引擎是 OpenClaw 的自我改进系统，通过 Gene（基因）机制沉淀知识和最佳实践，实现 Agent 的持续进化。

---

## 核心概念

### Gene（基因）
Gene 是可复用的知识单元，分为三类：

| 类型 | 前缀 | 适用范围 | 示例 |
|------|------|---------|------|
| **全局 Gene** | `gene_global_` | 所有 Agent | 系统架构、安全原则 |
| **共享 Gene** | `gene_shared_` | 指定 Agent 组 | 协作协议、记忆共享 |
| **专属 Gene** | `gene_<agent>_` | 单个 Agent | 角色定位、技能权限 |

### Gene 结构
```
~/.openclaw/genes/
├── gene_global_001.md    # 核心系统配置
├── gene_global_002.md    # 故障预警与运维
├── gene_shared_001.md    # 记忆共享协议
└── gene_media_001.md     # 小媒专属配置
```

## 命令参考

### 基础命令

```bash
# 列出所有 Gene
evolve list

# 读取指定 Gene
evolve read gene_global_001

# 应用 Gene 到当前 Agent
evolve apply gene_shared_001

# 验证 Gene 完整性
evolve verify

# 备份所有 Gene
evolve backup
```

### 记忆命令

```bash
# 搜索共享记忆
evolve memory search "关键词"

# 同步记忆到指定 Agent
evolve memory sync media

# 列出共享记忆
evolve memory list

# 整合所有 Agent 记忆
evolve memory integrate
```

## 使用场景

### 场景 1: 新 Agent 初始化
```bash
# 1. 读取全局配置
evolve read gene_global_001
evolve read gene_global_002

# 2. 应用共享协议
evolve apply gene_shared_001

# 3. 同步记忆
evolve memory sync <new-agent>
```

### 场景 2: 知识更新
```bash
# 1. 备份当前 Gene
evolve backup

# 2. 编辑 Gene 文件
vim ~/.openclaw/genes/gene_global_001.md

# 3. 验证完整性
evolve verify

# 4. 同步到所有 Agent
for agent in writer media hunter; do
    evolve memory sync $agent
done
```

### 场景 3: 故障排查
```bash
# 读取故障预警 Gene
evolve read gene_global_002

# 按指南执行排查步骤
```

## Gene 编写规范

### 文件格式
```markdown
# Gene <类型> <编号> - <标题>

**类型**: 全局/共享/专属
**版本**: x.x.x
**创建时间**: YYYY-MM-DD
**适用范围**: <适用对象>

---

## 章节 1

内容...

## 章节 2

内容...

---

*Gene 固化时间：YYYY-MM-DD*
```

### 命名规则
- 全局 Gene: `gene_global_NNN.md`
- 共享 Gene: `gene_shared_NNN.md`
- 专属 Gene: `gene_<agent>_NNN.md`

## 共享记忆

### 目录结构
```
~/.openclaw/agents/shared-memory/
├── system-status/        # 系统状态快照
├── cross-agent/          # 跨 Agent 协作
├── insights/             # 关键洞察
└── ontology/             # 实体关系
```

### 同步机制
- **自动同步**: 每天 3:00 AM
- **手动同步**: `evolve memory sync <agent>`
- **触发脚本**: `memory-consolidate.sh`

---

## 最佳实践

1. **定期备份**: 重要修改前执行 `evolve backup`
2. **版本控制**: Gene 文件纳入 Git 管理
3. **渐进更新**: 小步快跑，避免大规模变更
4. **验证优先**: 修改后执行 `evolve verify`
5. **文档同步**: 更新 Gene 后同步到相关 Agent

## 故障排查

### Gene 读取失败
```bash
# 检查文件是否存在
ls -la ~/.openclaw/genes/

# 检查权限
chmod 644 ~/.openclaw/genes/*.md
```

### 记忆同步失败
```bash
# 检查目标 Agent 目录
ls -la ~/.openclaw/agents/<agent>/

# 手动复制
mkdir -p ~/.openclaw/agents/<agent>/shared-memory
cp -r ~/.openclaw/agents/shared-memory/* \
      ~/.openclaw/agents/<agent>/shared-memory/
```
