## 🕐 14:00 - 小媒的制作计划

### 方案选择

**最终方案：HTML Canvas 精细版 + 墨墨协助**

**原因：**
1. 小媒无法直接操作 Figma/Canva（需要人工）
2. baoyu-image-gen 文字不可控
3. HTML Canvas 可以 100% 控制文字和布局

### 技术路线

1. **使用 Puppeteer/Playwright** - 浏览器自动化截图
2. **精细 HTML/CSS 布局** - 精确控制每个元素
3. **导入 SVG 图标** - 从 iconfont 或 FontAwesome
4. **正确配色** - 浅灰白 + 灰绿 + 粉色警示
5. **正确字体** - 黑体 + 宋体 + 等宽字体

### 需要墨墨帮助的地方

1. **安装 Puppeteer** - npm install puppeteer
2. **安装中文字体** - 检查系统是否有中文字体
3. **权限问题** - 可能需要墨墨批准某些命令
