## 🧠 QMD 记忆检索（按需使用）⭐ 新增

**定位**：辅助工具，按需使用，非强制检查项

### 使用场景

| 场景 | 触发条件 | 命令 |
|------|---------|------|
| **用户询问记忆** | 哥哥问"我之前说过 XXX 吗？" | `qmd search "关键词" -n 5` |
| **跨文件检索** | 需要查找多个记忆文件中的信息 | `qmd search "主题" -n 10` |
| **记忆巩固前置** | 每天 3:00 AM 记忆巩固前 | `qmd search "关键洞察" -n 5` |
| **项目回顾** | 需要回顾某个项目的历史决策 | `qmd search "项目名 决策" -n 5` |

### 快速命令参考

```bash
# 基础搜索（推荐，<1 秒响应）
qmd search "关键词" -n 5

# 多关键词搜索
qmd search "小媒 定位" -n 5

# 查看索引状态
qmd status
```

### 注意事项

- ⚠️ **避免使用语义搜索** (`qmd query`) - CPU 模式下极慢（30+ 分钟）
- ✅ **推荐 BM25 搜索** (`qmd search`) - 关键词匹配，毫秒级响应
- ✅ **索引范围**：memory/ 目录 + MEMORY.md + knowledge/ 目录
- ✅ **更新索引**：添加新文件后执行 `qmd update`

### 与现有记忆系统配合

```
QMD BM25 搜索（快速检索）
    ↓
定位到具体文件/行
    ↓
读取完整内容（feishu_fetch_doc 或 read 工具）
    ↓
整合到回复中
```
