## 技术教训

### Crontab 配置问题

**问题**: crontab 命令在非 TTY 模式下卡住

**现象**:
- 3 次尝试都卡住（session 被 SIGTERM）
- 需要 process kill 强制终止
- 最终系统通知显示成功

**根因**: crontab 命令需要交互式 TTY

**解决方案**:
1. ✅ 使用 `pty=true` 参数（已验证有效）
2. ✅ 或使用 launchd 替代（更可靠）

**命令示例**:
```bash
# 方式 1: pty=true
exec(command="crontab -l", pty=true)

# 方式 2: launchd（推荐）
launchctl load ~/Library/LaunchAgents/ai.openclaw.memory-consolidate.plist
```
