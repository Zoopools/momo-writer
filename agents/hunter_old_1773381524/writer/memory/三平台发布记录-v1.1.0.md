## 🚀 发布流程

### 阶段 1: 重新打包

```bash
cd ~/Documents/openclaw/agents/writer/config-templates/

# 查看文件列表
ls -la
# SKILL.md
# 配置模板合集（脱敏版）.md
# AGENTS.md 配置模板.md
# 记忆系统配置模板.md
# 多 Agent 配置模板.md

# 重新打包
rm -f /tmp/config-center-v1.1.0.zip
zip -r /tmp/config-center-v1.1.0.zip .
```

### 阶段 2: 发布到水产市场

```bash
DEVICE_ID=$(cat ~/.openclaw/identity/device.json | jq -r .deviceId)

curl -X POST "https://openclawmp.cc/api/v1/assets/publish" \
  -H "X-Device-ID: $DEVICE_ID" \
  -F "package=@/tmp/config-center-v1.1.0.zip" \
  -F 'metadata={
    "name": "OpenClaw 集中配置管理系统",
    "type": "skill",
    "version": "1.1.0",
    "description": "为 OpenClaw 构建集中化配置管理系统，告别硬编码和配置分散，实现改一处生效全局的现代化运维体验。包含配置加载器、主配置融合、记忆同步、AGENTS.md 模板、memoryFlush、memorySearch、多 Agent 配置等核心功能。",
    "author": "墨墨 (Mò)",
    "tags": ["配置管理", "运维工具", "模块化", "配置中心", "记忆系统", "多 Agent"],
    "min_version": "2026.3.2"
  }' | jq '.'
```

### 阶段 3: 发布到 ClawHub

```bash
clawhub publish \
  --slug "openclaw-config-center" \
  --name "OpenClaw 集中配置管理系统" \
  --version "1.1.0" \
  --changelog "v1.1.0: 新增 AGENTS.md 配置模板、记忆系统配置模板（memoryFlush+memorySearch）、多 Agent 配置模板（子 Agent+ 独立 Agent）。基于进阶配置教程和多 Agent 配置指南深度优化。" \
  --tags "config,management,modular,openclaw,chinese,memory,multi-agent" \
  ~/Documents/openclaw/agents/writer/config-templates/
```
