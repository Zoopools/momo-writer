## 📋 完整工作流（7 步）

### 步骤 1: 创作文章（Markdown）

**技能**: 小媒 LLM 能力

**输出**: `/tmp/article.md`

```markdown
---
title: 文章标题
author: 哥哥
---

# 文章正文

这里是内容...
```

**要点**:
- ✅ 使用 YAML Frontmatter（标题、作者）
- ✅ Markdown 格式
- ✅ 结构清晰（H1/H2/H3）

---

### 步骤 2: 生成封面图

**技能**: `baoyu-cover-image`

**命令**:
```bash
bun ~/.openclaw/agents/media/skills/baoyu-skills/skills/baoyu-cover-image/scripts/generate-cover.ts \
  --title "文章标题" \
  --output /tmp/cover.jpg
```

**参数**:
- `--title`: 文章标题
- `--output`: 封面图输出路径
- `--aspect`: 比例（默认 2.35:1 电影宽屏）

**输出**: `/tmp/cover.jpg`（2.35:1 比例）

---

### 步骤 3: Markdown 格式化

**技能**: `baoyu-format-markdown`

**命令**:
```bash
bun ~/.openclaw/agents/media/skills/baoyu-skills/skills/baoyu-format-markdown/scripts/format.ts \
  --input /tmp/article.md \
  --output /tmp/article-formatted.md
```

**功能**:
- ✅ 添加 Frontmatter
- ✅ 格式化标题
- ✅ 优化段落
- ✅ 添加摘要

---

### 步骤 4: 转换为微信 HTML

**技能**: `md-to-wechat`（内置）

**命令**（在 wechat-article.ts 中自动执行）:

```bash
bun ~/.openclaw/agents/media/skills/baoyu-skills/skills/baoyu-post-to-wechat/scripts/wechat-article.ts \
  --markdown /tmp/article-formatted.md \
  --cover /tmp/cover.jpg \
  --profile chrome \
  --target-id <标签页 ID> \
  --submit
```

**功能**:
- ✅ Markdown → 微信 HTML
- ✅ 应用主题/配色
- ✅ 插入封面图
- ✅ 格式化代码块

---

### 步骤 5: 检查标签页状态

**命令**:
```bash
openclaw browser --browser-profile chrome tabs
```

**预期输出**:
```
1. 公众号
   https://mp.weixin.qq.com/cgi-bin/home?t=home/index&lang=zh_CN&token=xxx
   id: 1D0B45C65218CE57CD5A2DEE27F42B72
```

**如果没有标签页**:
- ❌ 提醒哥哥：「哥哥，请附加公众号后台标签页」
- ⏸️ 等待哥哥附加后再继续

---

### 步骤 6: Chrome 扩展发布

**核心命令**:
```bash
bun ~/.openclaw/agents/media/skills/baoyu-skills/skills/baoyu-post-to-wechat/scripts/wechat-article.ts \
  --markdown /tmp/article-formatted.md \
  --cover /tmp/cover.jpg \
  --profile chrome \
  --target-id 1D0B45C65218CE57CD5A2DEE27F42B72 \
  --submit
```

**参数说明**:

| 参数 | 值 | 说明 |
|------|-----|------|
| `--markdown` | `/tmp/article-formatted.md` | 格式化后的文章 |
| `--cover` | `/tmp/cover.jpg` | 封面图 |
| `--profile` | `chrome` | **使用 Chrome 扩展** |
| `--target-id` | `1D0B45C65218CE57CD5A2DEE27F42B72` | **公众号标签页 ID** |
| `--submit` | (无值) | **直接发布** |

**执行流程**:
1. ✅ 打开 Chrome（使用哥哥的登录状态）
2. ✅ 导航到公众号后台
3. ✅ 点击「新建图文」
4. ✅ 填写标题
5. ✅ 粘贴 HTML 内容
6. ✅ 上传封面图
7. ✅ 点击「发表」
8. ✅ 确认群发

## 🚀 一键执行脚本

**位置**: `~/.openclaw/agents/media/scripts/publish-wechat.sh`

```bash
#!/bin/bash
set -e

# 参数
MARKDOWN_FILE="${1:-/tmp/article.md}"
TITLE=$(grep "^title:" "$MARKDOWN_FILE" | cut -d':' -f2 | tr -d ' "')

echo "╔════════════════════════════════════════╗"
echo "║   微信公众号发布流程                   ║"
echo "╚════════════════════════════════════════╝"
echo ""
echo "文章：$MARKDOWN_FILE"
echo "标题：$TITLE"
echo ""

# 步骤 1: 生成封面图
echo "[1/5] 生成封面图..."
bun ~/.openclaw/agents/media/skills/baoyu-skills/skills/baoyu-cover-image/scripts/generate-cover.ts \
  --title "$TITLE" \
  --output /tmp/cover.jpg

# 步骤 2: 格式化 Markdown
echo "[2/5] 格式化 Markdown..."
bun ~/.openclaw/agents/media/skills/baoyu-skills/skills/baoyu-format-markdown/scripts/format.ts \
  --input "$MARKDOWN_FILE" \
  --output /tmp/article-formatted.md

# 步骤 3: 检查标签页
echo "[3/5] 检查标签页..."
openclaw browser --browser-profile chrome tabs

# 步骤 4: 发布
echo "[4/5] 发布到公众号..."
bun ~/.openclaw/agents/media/skills/baoyu-skills/skills/baoyu-post-to-wechat/scripts/wechat-article.ts \
  --markdown /tmp/article-formatted.md \
  --cover /tmp/cover.jpg \
  --profile chrome \
  --target-id 1D0B45C65218CE57CD5A2DEE27F42B72 \
  --submit

# 步骤 5: 通知
echo "[5/5] 通知哥哥..."
echo "✅ 发布完成！"

echo ""
echo "╔════════════════════════════════════════╗"
echo "║   发布成功！                           ║"
echo "╚════════════════════════════════════════╝"
```

**使用方式**:
```bash
bash ~/.openclaw/agents/media/scripts/publish-wechat.sh /tmp/article.md
```

## ⚠️ 常见问题

### 问题 1: 忘记使用 `--profile chrome`

**错误**:
```bash
# ❌ 这会使用独立浏览器，需要重新登录
bun wechat-article.ts --markdown article.md
```

**正确**:
```bash
# ✅ 使用 Chrome 扩展，复用登录状态
bun wechat-article.ts --markdown article.md --profile chrome --target-id <ID>
```

---

### 问题 2: 标签页未附加

**错误**:
```
Error: No tabs found for profile: chrome
```

**解决**:
1. 提醒哥哥附加标签页
2. 等待哥哥完成后重试

---

### 问题 3: 今天群发次数已用完

**错误**:
```
Error: 今天群发次数已用完
```

**解决**:
- ✅ 文章保存到草稿箱
- ✅ 明天再群发
- ✅ 订阅号每天 1 次

---

### 问题 4: 封面图上传失败

**错误**:
```
Error: 封面图上传失败
```

**解决**:
1. 检查封面图格式（JPG/PNG）
2. 检查文件大小（<2MB）
3. 重新生成封面图
