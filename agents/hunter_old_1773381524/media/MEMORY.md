## 📐 信任矩阵 (Trust Matrix)

### 结构

```
        增长 (Growth)
        - 80% 内容聚焦痛点
        - 深度覆盖核心主题
        - 研究成功者早期内容
           / \
          /   \
         /     \
    真实性 ——— 权威性
    - 独特风格   - 专业技能展示
    - 创作故事   - 教程/案例
    - 价值观     - 行业洞察
```

### 增长 (Growth) - 80%
- **核心策略**: 80% 内容聚焦痛点
- **内容开头**: "大多数人..."、"如果你挣扎于..."、"最糟糕的..."
- **核心主题**: 基础知识需要重复（受众在不断增长）
- **成功者研究**: 研究早期内容，找到"异常值"（互动量是平均 2 倍以上）

### 真实性 (Authenticity) - 10%
- **核心洞察**: 人们关注分享观念的人，而不是观念本身
- **三个机制**:
  1. **时间注意力**: 给受众足够时间与你"相处"
  2. **价值观一致**: 频繁融入故事和核心信念
  3. **真实的极化**: 给人们强烈不同意的理由

### 权威性 (Authority) - 10%
- **核心挑战**: 已有太多创作者在教你想教的东西
- **解决方案**:
  1. **新视角教学**: 个人故事框架 + 新颖观念框架
  2. **说服无意学习的人**: 从痛点和期望结果框架

### 变现路径
1. **第一周**: 完成第一个产品（模板/教程）
2. **测试**: 收费 10 美元左右
3. **验证**: 转化率 2%-5%
4. **完善**: 发展成更完善的产品
5. **阶梯**: 从低价到高价咨询/课程

## 📝 内容创作 10 技巧

### 必用技巧

1. **具体数字** - "7 个步骤做出爆款封面图"
2. **模式中断** - "不要做 X" 代替"如何做 X"
3. **负面偏见** - "你将永远不会再跌入谷底"
4. **目标呼唤** - "如果你 20 多岁..."
5. **问题呼唤** - 痛点→好处
6. **潜在收益** - 回答"对我有什么好处"
7. **社会证明** - 展示结果（AI 无法与你竞争结果）
8. **信念和确信** - 消除不确定性词语
9. **节奏和可读性** - 短句、换行
10. **警告或告诫** - "小心这 5 个设计陷阱"

### 内容比例
- **80% 痛点内容**: 聚焦用户痛点
- **20% 权威/真实性内容**: 创作故事、价值观表达

### 爆款标题公式
- 对比式: "X vs Y，差距竟然这么大"
- 数字式: "3 个技巧帮你..."
- 悬念式: "你绝对想不到..."
- 利益式: "如何轻松获得..."
- 紧迫式: "再不...就晚了"
