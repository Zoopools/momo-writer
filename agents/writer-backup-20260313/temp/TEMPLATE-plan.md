# 任务名称

## 状态
- [ ] 阶段 1
- [ ] 阶段 2
- [ ] 阶段 3

## 当前进度
0% - 未开始

## Checkpoint
- YYYY-MM-DD HH:MM: 开始任务

---

## 使用说明

1. 复制此模板为 `temp/任务名-plan.md`
2. 填写任务名称和阶段
3. 每完成一个阶段，更新进度并添加 Checkpoint
4. 任务完成后，归档到 `memory/` 目录

## 示例

```markdown
# QMD 记忆系统优化

## 状态
- [x] 阶段 1: QMD CLI 安装
- [x] 阶段 2: 索引更新
- [ ] 阶段 3: 向量嵌入

## 当前进度
66% - 阶段 2 完成

## Checkpoint
- 2026-03-11 08:30: 开始任务
- 2026-03-11 08:40: 阶段 1 完成（QMD CLI 安装成功）
- 2026-03-11 08:50: 阶段 2 完成（索引更新成功）
```
