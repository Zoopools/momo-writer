### ClawHub 发布教训（重要！）

**问题**: CLI 有 `acceptLicenseTerms` bug，无法发布

**解决方案**: 使用网页发布

**浏览器自动化教训**:
- ❌ `fill` 动作 - 只设置 value 属性，不触发 JS 事件
- ✅ `type` 动作 - 模拟真实键盘输入，触发 input/change 事件

**正确做法**:
```javascript
// 逐个字段 type
browser.action.act({ kind: "type", ref: "e46", text: "openclaw-sandbox" })
browser.action.act({ kind: "type", ref: "e48", text: "OpenClaw 沙盒测试系统" })
```

**已沉淀到**: MEMORY.md → ClawHub 发布教训章节
