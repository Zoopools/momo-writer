```bash

# 示例：创建"小宣"（宣传专家）

  xiaoxuan \
```

## 🛡️ 备份与恢复

### 备份

```bash
# 自动备份（执行架构改造前）
~/.openclaw/backups/pre-architecture-YYYYMMDD_HHMMSS/
```

### 一键恢复

```bash
# 恢复最新备份
bash ~/.openclaw/backups/restore.sh

# 恢复指定备份
bash ~/.openclaw/backups/restore.sh ~/.openclaw/backups/pre-architecture-YYYYMMDD_HHMMSS
```
