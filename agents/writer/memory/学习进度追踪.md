**2. Twitter 技术推文**（哥哥分享）✅ 已学习
- **链接：** https://x.com/kevinma_dev_zh/status/2028369395517726932
- **来源：** @kevinma_dev_zh
- **主题：** 让 Claude Cowork 效率提升 100 倍的 17 个 Best Practices
- **核心内容：**
  1. **三层文件组织结构** (_MANIFEST.md)
     - Tier 1: Canonical（品牌规范、项目简报、当前策略）
     - Tier 2: Domain（主题子文件夹）
     - Tier 3: Archival（旧草稿、参考资料）
  2. **Global Instructions 模板**
     - 在 Settings → Cowork → Edit → Global Instructions 设置
     - 优先读取_MANIFEST.md
     - 执行前先提问澄清
     - 质量标准：无需编辑即达客户可用级别
  3. **Context 文件夹 (00_Context)**
     - about-me.md - 职业身份
     - brand-voice.md - 沟通风格
     - working-style.md - 工作方式
  4. **任务设计原则**
     - 告诉"完成是什么样子"而非"怎么做"
     - 每个任务回答三个问题：完成什么样？约束条件？不确定时怎么做？
     - 合并相关任务，一次运行多个 deliverable
  5. **Subagent 使用**
     - 并行处理复杂任务
     - 适用：竞争分析、多源研究、批量文件处理
  6. **Uncertainty Handling**
     - 如果日期不清晰，标记为 VERIFY
     - 如果文件可能属于多个文件夹，放入/needs-review
     - 如果分类把握不足 80%，标记而不是猜测
- **对墨墨的启发：**
