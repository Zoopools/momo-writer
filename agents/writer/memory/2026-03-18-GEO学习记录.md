## 📚 今日学习核心内容

### 1. GEO 基础概念

**定义**: GEO (Generative Engine Optimization) = 生成式引擎优化
- 针对 AI 搜索平台（ChatGPT、Perplexity、文心一言、豆包等）优化内容
- 让 AI 在回答用户问题时**检索、引用并推荐**品牌
- 与 SEO 区别：SEO 争排名，GEO 争 AI 引用

**核心公式**:
```
内容满足 AI 搜索需求 + 充足的高质量信息源 = 高概率被 AI 采纳引用
```

### 3. GEO 展现形式（6种）

1. **直接答案生成** - AI 直接给出品牌推荐列表
2. **引用式回答** - 答案带 [1][2][3] 来源标注
3. **对比式总结** - 表格形式对比多个品牌
4. **结构化数据** - 知识卡片展示品牌信息
5. **推荐列表** - 类似导购的编辑推荐
6. **问答式引用** - 自然融入品牌提及

### 6. 平台生态隔离

**关键洞察**:
> 在豆包做GEO，百度看不到；在百度做GEO，豆包也看不到。

**两个独立战场**:
```
字节生态（豆包）          百度生态（文心一言）
├── 抖音                  ├── 百度搜索
├── 今日头条              ├── 百度知道
└── 西瓜视频              ├── 百家号
    ↓ 不通                    ↓ 不通
【豆包AI摘要】            【百度AI摘要】
```
