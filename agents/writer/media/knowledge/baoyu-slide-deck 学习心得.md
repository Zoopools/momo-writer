## 📚 技能概览

**baoyu-slide-deck** 是一个专业的幻灯片生成技能，可以从内容自动生成高质量幻灯片图片。

### 核心特点
- **16 种预设风格**：从 blueprint 到 vintage，覆盖教育、企业、科技、创意等多种场景
- **5 维度风格系统**：Texture + Mood + Typography + Density，可组合出 450 种独特风格
- **完整工作流**：9 步流程确保质量和用户满意度
- **智能内容分析**：自动识别主题、受众、语言、推荐风格

## 💡 核心洞察

### 1. 设计哲学：阅读优先 vs 演讲优先

传统幻灯片用于现场演讲，需要演讲者口头解释。而 baoyu-slide-deck 生成的幻灯片：
- ✅ 每页必须自解释，无需口头评论
- ✅ 逻辑流畅，所有上下文都在幻灯片内
- ✅ 优化社交媒体分享（可独立传播）

这个理念对新媒体运营非常有价值：
- 可将长文章拆分为幻灯片系列
- 在小红书、朋友圈、微博等平台传播
- 每页都是独立的"知识卡片"

### 2. 风格系统的威力

**16 种预设风格**已经覆盖了大部分场景：

| 场景 | 推荐风格 | 理由 |
|------|---------|------|
| 技术教程 | blueprint | 网格 + 冷色调 + 技术字体 |
| 教育培训 | sketch-notes | 手绘 + 温暖 + 手写字体 |
| 企业汇报 | corporate | 干净 + 专业 + 几何字体 |
| 高管简报 | minimal | 干净 + 中性 + 极简 |
| 产品发布 | bold-editorial | 高对比 + 杂志感 |
| 数据分析 | notion | SaaS 风格 + 密集数据 |
| 生活方式 | watercolor | 水彩 + 温暖 + 人文字体 |

**5 维度自定义**让创意无限：
- Texture（纹理）：clean、grid、organic、pixel、paper
- Mood（情绪）：professional、warm、cool、vibrant、dark、neutral
- Typography（字体）：geometric、humanist、handwritten、editorial、technical
- Density（密度）：minimal、balanced、dense

### 3. 工作流的严谨性

9 步流程看似复杂，但每一步都有明确目的：

```
1. 设置 & 分析 → 理解内容、检测语言、推荐风格
2. 确认（两轮）→ 风格、受众、幻灯片数量、审查选项
3. 生成大纲 → 每页幻灯片的叙事和视觉描述
4. 审查大纲（条件性）→ 确保结构和逻辑
5. 生成提示词 → 转换为 AI 图像生成提示
6. 审查提示词（条件性）→ 确保画面质量
7. 生成图片 → 逐页生成幻灯片
8. 合并 PPTX/PDF → 输出可编辑格式
9. 输出总结 → 完整报告
```

关键点：
- **第 2 步确认必须执行**，避免生成不满意的幻灯片
- 第 4、6 步根据用户选择跳过，平衡效率和精度
- 支持部分工作流（--outline-only、--prompts-only、--images-only）

### 4. 内容规则的颠覆性

**叙事性标题** vs 标签式标题：

| ❌ 标签式 | ✅ 叙事式 |
|---------|---------|
| "关键数据" | "使用量在 6 个月内翻倍" |
| "我们的解决方案" | "一个平台替代五个工具" |
| "核心优势" | "团队每周节省 10 小时" |

叙事式标题：
- 直接传达核心价值
- 不依赖上下文
- 更有传播力

**避免 AI 陈词滥调**：
- ❌ "Let's explore"、"Dive into"
- ❌ "Exciting"、"Amazing"、"Revolutionary"
- ❌ "In conclusion"、"To summarize"

这些词让内容显得机械、不真诚。

### 5. 视觉机会映射

不是所有内容都需要可视化，也不是所有内容都不需要：

| 内容类型 | 视觉处理 | 例子 |
|---------|---------|------|
| 对比 | 并排比较 | 功能对比表 |
| 流程 | 流程图、编号步骤 | 工作流程图 |
| 层次 | 组织结构图、金字塔 | 组织架构 |
| 时间线 | 水平/垂直时间线 | 项目里程碑 |
| 数据 | 图表、突出数字 | 关键指标 |
| 概念 | 图标、隐喻、插图 | 抽象想法可视化 |

## 🎯 与新媒体运营的结合

### 1. 内容形态扩展

**长文章 → 幻灯片系列**：
- 每页幻灯片是一个"知识点"
- 在小红书、朋友圈等平台发布
- 降低阅读门槛，提高传播率

**产品介绍 → 视觉化演示**：
- 产品功能拆分为 10-15 页幻灯片
- 每页突出一个核心价值
- 适合快速浏览和分享

**数据分析 → 信息图幻灯片**：
- 关键数据用幻灯片展示
- 每页一个关键指标
- 适合向领导汇报

### 2. 平台适配策略

| 平台 | 幻灯片策略 |
|------|-----------|
| 小红书 | 1-3 张幻灯片，封面大标题，内容分点 |
| 公众号 | 嵌入幻灯片，配文字说明 |
| 微博 | 单张幻灯片 + 短文案 |
| 朋友圈 | 单张幻灯片，配文字分享 |

### 3. 风格选择建议

**小红书**：
- `sketch-notes`：手绘风，亲和力强
- `watercolor`：水彩风，艺术感
- `minimal`：极简风，高级感

**企业内容**：
- `corporate`：企业风，专业可信
- `notion`：SaaS 风格，数据驱动

**科技内容**：
- `blueprint`：技术蓝图，专业严谨
- `intuition-machine`：技术文档，学术风
