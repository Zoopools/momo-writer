## 🔍 当前可用工具状态

### 1. Web Search (Brave API) - ⚠️ 需要配置

**工具名称**: `web_search`  
**类型**: 内置工具  
**状态**: ❌ **缺少 API Key，无法使用**

**错误信息**:
```
web_search (brave) needs a Brave Search API key.
Run `openclaw configure --section web` to store it,
or set BRAVE_API_KEY in the Gateway environment.
```

**配置方法**:
```bash
# 方法1: 使用 OpenClaw 配置
openclaw configure --section web

# 方法2: 设置环境变量
export BRAVE_API_KEY=your_api_key_here
```

**功能**:
- Brave Search API 搜索
- 支持多语言、地区过滤
- 支持时间范围过滤

---

### 2. Web Fetch - ⚠️ 部分可用

**工具名称**: `web_fetch`  
**类型**: 内置工具  
**状态**: ⚠️ **有限可用**

**限制**:
- 某些网站会返回 `Blocked: resolves to private/internal/special-use IP address`
- 无法访问需要 JavaScript 渲染的页面

**适用场景**:
- 简单的静态网页抓取
- API 端点请求
- 文档下载

---

### 3. Browser (Chrome CDP) - ✅ 可用

**工具名称**: `browser`  
**类型**: 内置工具 + chrome-cdp Skill  
**状态**: ✅ **正常运行**

**功能**:
- 控制 Chrome 浏览器
- 页面截图、快照
- 点击、输入等交互操作
- 页面导航

**使用示例**:
```javascript
// 打开页面
browser({action: "open", url: "https://google.com"})

// 获取页面快照
browser({action: "snapshot", targetId: "..."})

// 点击元素
browser({action: "act", targetId: "...", request: {kind: "click", ref: "e1"}})
```

**Skill 位置**: `~/.openclaw/skills/chrome-cdp/`

---

### 4. X Tweet Fetcher - ⚠️ 需要 Camofox

**工具名称**: `x-tweet-fetcher`  
**类型**: 本地工具 (Python 脚本)  
**状态**: ⚠️ **部分功能可用，高级功能需要 Camofox**

**位置**: `~/.openclaw/tools/x-tweet-fetcher/`

**功能分级**:

| 功能 | 依赖 | 状态 |
|------|------|------|
| 单条推文抓取 | 无 | ✅ 可用 |
| 回复线程 | Camofox | ❌ 需要安装 |
| 用户时间线 | Camofox | ❌ 需要安装 |
| 中文平台抓取 | Camofox | ❌ 需要安装 |
| **Google 搜索** | Camofox | ❌ 需要安装 |

**Camofox 搜索功能**:
```python
from scripts.camofox_client import camofox_search
results = camofox_search("your query")
# 返回: [{"title": "...", "url": "...", "snippet": "..."}, ...]
```

**Camofox 安装**:
```bash
# 方法1: OpenClaw 插件
openclaw plugins install @askjo/camofox-browser

# 方法2: 手动安装
git clone https://github.com/jo-inc/camofox-browser
cd camofox-browser
npm install && npm start

# 验证
curl http://localhost:9377/health
# 应返回: {"status":"ok"}
```

## 🔧 推荐配置方案

### 方案 A: 完整功能（推荐）

**步骤**:
1. **配置 Brave API Key**
   ```bash
   openclaw configure --section web
   # 输入 Brave Search API Key
   ```

2. **安装 Camofox**（可选，用于社交媒体抓取）
   ```bash
   openclaw plugins install @askjo/camofox-browser
   ```

**优点**: 所有搜索功能可用
**缺点**: Brave API 需要付费（有免费额度）

### 方案 B: 免费方案

**使用**:
- `browser` 工具进行网页抓取和搜索
- `x-tweet-fetcher` + Camofox 进行 Google 搜索
- `web_fetch` 进行简单 API 请求

**优点**: 完全免费
**缺点**: 需要更多步骤，不如 Brave API 方便

## 📝 当前实际可用方案

### 立即可以使用的搜索方式

1. **Browser 工具**（最可靠）
   ```javascript
   // 打开 Google 搜索
   browser({action: "open", url: "https://google.com/search?q=关键词"})
   
   // 获取结果
   browser({action: "snapshot", targetId: "..."})
   ```

2. **X Tweet Fetcher 基础功能**
   ```bash
   # 抓取单条推文（无需 Camofox）
   python3 ~/.openclaw/tools/x-tweet-fetcher/scripts/fetch_tweet.py \
     --url "https://x.com/user/status/123456"
   ```

3. **飞书文档搜索**
   ```javascript
   feishu_search_doc_wiki({action: "search", query: "关键词"})
   ```

4. **本地记忆搜索**
   ```javascript
   memory_search({query: "关键词"})
   ```

### 需要配置后才能使用的

1. **Brave Web Search** - 需要 API Key
2. **Camofox 搜索** - 需要安装 Camofox 服务

## 🎯 给哥哥的建议

**当前状况**:
- 我们有 browser 工具可以完成搜索任务（通过 Google）
- web_search (Brave) 需要配置 API Key
- x-tweet-fetcher 的搜索功能需要 Camofox

**是否需要配置 Brave API？**

| 场景 | 建议 |
|------|------|
| 偶尔搜索 | 使用 browser 工具即可 |
| 频繁搜索 | 配置 Brave API，更快速方便 |
| 社交媒体抓取 | 安装 Camofox |

**墨墨的建议**:
如果哥哥需要频繁进行网络搜索，建议配置 Brave API Key，这样可以：
- 更快速地获取搜索结果
- 支持更多过滤选项（时间、语言、地区）
- 避免浏览器操作的延迟

如果只是偶尔搜索，现在的 browser 工具完全够用～🖤
