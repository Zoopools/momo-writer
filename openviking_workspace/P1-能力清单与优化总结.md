## 自动化配置

### Crontab 任务

```bash
# 健康检查（每 30 分钟）
*/30 * * * * ~/.openclaw/scripts/gateway-monitor.sh

# 早报（每天 8:00）
0 8 * * * ~/.openclaw/scripts/morning-briefing.sh

# 记忆巩固（每天 3:00）
0 3 * * * ~/.openclaw/scripts/memory-consolidate.sh

# 日志清理（每周日 3:00）
0 3 * * 0 find ~/.openclaw/logs -name "*.log" -mtime +7 -delete
```

## 故障预案

### 常见故障

| 故障 | 原因 | 解决方案 |
|------|------|---------|
| 400 错误 | Token 超限 | 截断输入 <150k |
| Gateway 失联 | 进程崩溃 | 自动重启 |
| 飞书断开 | 网络问题 | 自动重连 |
| 记忆溢出 | 上下文过长 | Compaction |

### 紧急恢复

```bash
# 1. 恢复备份
bash ~/.openclaw/backups/restore.sh

# 2. 重启 Gateway
openclaw gateway restart

# 3. 验证状态
openclaw status
```
