# 小红书标题模板

## 爆款标题公式

### 1. 数字法
**公式**: 「X 个技巧/方法/步骤让你...」

**示例**:
- 「3 个技巧让你的笔记阅读量翻倍」
- 「5 个方法帮你快速涨粉 1000+」
- 「7 天涨粉 5000 的实操步骤」

### 2. 疑问法
**公式**: 「为什么你总是...？」

**示例**:
- 「为什么你的小红书总是没人看？」
- 「为什么你发了 100 篇笔记还是 0 粉？」
- 「为什么别人轻松涨粉你却不行？」

### 3. 对比法
**公式**: 「X vs Y，差距竟然...」

**示例**:
- 「月薪 3 千 vs 月薪 3 万的小红书运营差距」
- 「新手 vs 老手，这 5 个细节太真实了」
- 「免费 vs 付费，流量差距有多大？」

### 4. 情绪法
**公式**: 「情绪词 + 核心卖点」

**示例**:
- 「救命！这个太绝了」
- 「真的会谢！早该知道这个」
- 「破防了！原来这才是真相」

### 5. 痛点法
**公式**: 「如果你...一定要看」

**示例**:
- 「如果你笔记总是没人看，一定要看这篇」
- 「如果你发了很多篇还是 0 粉，这个方法救你」
- 「如果你不会写标题，直接套用这个公式」
