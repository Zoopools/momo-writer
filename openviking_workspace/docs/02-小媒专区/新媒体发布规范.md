# 新媒体发布规范

## 发布流程

```
内容准备 → 墨墨审查 → 平台选择 → 定时发布 → 数据跟踪
```

## 平台规范

### Twitter/X
- 图片尺寸：1200x675
- 文案长度：280字符
- 最佳时间：9:00-11:00, 19:00-21:00

### 小红书
- 图片尺寸：3:4 或 1:1
- 文案长度：1000字以内
- 话题标签：3-5个

### 抖音
- 视频尺寸：9:16
- 时长：15-60秒
- 封面：吸引眼球

### 微信公众号
- 封面尺寸：900x383
- 正文：图文混排
- 摘要：120字以内

## 发布检查

- [ ] 内容已通过墨墨审查
- [ ] 平台账号已登录
- [ ] 发布时间已确认
- [ ] 跟踪链接已配置
