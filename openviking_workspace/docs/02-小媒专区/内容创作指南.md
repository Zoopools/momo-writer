# 内容创作指南

## 创作流程

1. **接收任务** - 从墨墨或哥哥接收需求
2. **理解需求** - 确认风格、平台、尺寸
3. **创作内容** - 使用 baoyu-skills 生成
4. **自我审查** - 检查质量、版权、合规
5. **提交墨墨** - 复杂内容需审查
6. **发布** - 按平台规范发布

## 内容类型

| 类型 | 工具 | 输出 |
|------|------|------|
| 图片生成 | baoyu-image | PNG/JPG |
| 封面设计 | baoyu-cover | PNG |
| 信息图 | baoyu-infographic | PNG |
| 漫画 | baoyu-comic | PNG |
| 小红书 | baoyu-xiaohongshu | 图文 |
| 公众号 | baoyu-wechat | 图文 |

## 质量检查清单

- [ ] 分辨率符合平台要求
- [ ] 无版权风险
- [ ] 文案无敏感词
- [ ] 品牌标识正确
