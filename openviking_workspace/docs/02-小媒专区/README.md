## 📚 与主知识库的关系

### 小媒专区 (docs/02-小媒专区/)
- **定位**: 创意素材、模板、案例
- **内容**: 文案模板、封面设计、发布计划
- **特点**: 轻量、可复用、快速迭代

### 知识库 (knowledge/)
- **定位**: 深度指南、流程规范
- **内容**: 视频制作流程、脚本创作模板、平台发布指南
- **特点**: 系统、完整、长期维护

### 关系
```
小媒专区 (素材/模板) → 支撑 → 内容创作
    ↓
知识库 (流程/规范) → 指导 → 创作流程
```

## 🎨 素材库使用指南

### 文案模板

**用途**: 快速生成各平台文案

**包含**:
- 小红书标题模板
- 公众号文章模板
- 视频脚本模板
- 社交媒体文案模板

### 图片素材

**用途**: 封面图/信息图参考

**包含**:
- 优秀封面案例
- 信息图示例
- 配色方案参考

### 配色方案

**用途**: 品牌色彩规范

**包含**:
- 主色调 (#RRGGBB)
- 辅助色
- 强调色
- 应用场景

### 封面模板

**用途**: 各平台封面设计规范

**包含**:
- B 站封面模板 (1920x1080)
- 抖音封面模板 (1080x1920)
- 小红书封面模板 (1080x1440)
- 公众号封面模板 (900x383)

### 发布日历

**用途**: 内容发布计划

**包含**:
- 月度发布计划
- 重要节点标记
- 平台活动时间
- 热点日历
