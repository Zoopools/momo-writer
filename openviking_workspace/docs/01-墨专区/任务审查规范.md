## 三、审查标准

### 内容审查（小媒任务）

- [ ] **质量**：符合需求描述
- [ ] **版权**：无侵权风险
- [ ] **合规**：无敏感内容
- [ ] **品牌**：标识使用正确
- [ ] **格式**：符合平台规范

### 数据审查（小猎任务）

- [ ] **完整性**：数据无缺失
- [ ] **准确性**：来源可靠
- [ ] **时效性**：数据未过期
- [ ] **安全性**：无隐私泄露
- [ ] **格式**：符合分析要求

---

## 四、审查流程

### 通过标准

满足所有检查项 → ✅ **通过** → 交付哥哥

### 不通过处理

1. **标记问题**：明确说明不通过原因
2. **退回修改**：返回执行 Agent
3. **重新提交**：修改后再次审查
4. **最多 3 轮**：超过需升级处理

---

## 五、紧急任务

### 定义
- 时间敏感（< 2 小时）
- 影响重大（系统故障等）

### 流程
```
哥哥 → 墨墨（紧急模式）→ 立即执行 → 事后补审
```

### 事后补审
- 24 小时内完成审查
- 记录异常情况
- 优化流程避免重复
