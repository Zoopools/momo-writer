# 🔄 双端同步指南

## 同步机制

### 自动同步（Pulse Engine）
- **触发**：文件修改后 60 秒
- **动作**：自动 git commit & push
- **范围**：memory/, docs/, Gene/, 配置文件

### 周期同步（Patrol Engine）
- **触发**：每 5 分钟
- **动作**：git pull --rebase
- **功能**：获取对端更新

### 手动同步
```bash
# 立即执行同步
cd ~/Documents/openclaw/openviking_workspace
./scripts/sync-memory.sh
```

## 同步范围

```
Git 仓库 (momo-writer)
├── docs/              ✅ 知识库（实时同步）
├── memory/           ✅ 每日记忆（实时同步）
├── Gene/             ✅ Gene 文件（手动同步）
├── ~/.media/         ❌ 密钥（Git 忽略）
└── 会话历史          ❌ AGFS 同步（独立）
```

## 冲突解决

```bash
# 如遇冲突，以本地为准
git checkout --ours <文件>
git add <文件>
git commit -m "resolve: 冲突解决"
```
