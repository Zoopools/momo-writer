## 📝 使用规范

### 文件头部标签（YAML）
每个 Markdown 文件必须包含：

```yaml
---
agent: momo        # 负责人：momo / media / hunter
category: architecture  # 分类：architecture / media / tools / log
status: stable     # 状态：draft / stable / deprecated
date: 2026-03-16   # 创建日期
---
```

### 命名规范
- **目录**：`00-名称/`（数字前缀，小写，连字符）
- **文件**：`文件名称.md`（小写，连字符分隔）
- **图片**：`YYYY-MM-DD_描述.png`

### 编辑流程
1. **本地编辑**：Obsidian 修改
2. **自动同步**：`sync-memory.sh` 5分钟自动提交
3. **在线查看**：GitHub Pages 自动部署
4. **双端同步**：另一设备 `git pull` 获取更新
