# 数据采集手册

## 采集流程

1. **明确需求** - 确认采集目标、平台、时间范围
2. **选择工具** - 根据平台选择合适工具
3. **配置认证** - 通过墨墨获取 Cookie/Token
4. **执行采集** - 运行采集命令
5. **数据清洗** - 格式化、去重、筛选
6. **存储归档** - 保存到情报档案
7. **分析报告** - 生成情报分析报告

## 工具选择指南

| 平台 | 推荐工具 | 命令 |
|------|---------|------|
| Twitter/X | x-tweet-fetcher | `fetch-tweet --url` |
| 微博/B站/CSDN | x-tweet-fetcher | `fetch_china.py` |
| Reddit | Agent-Reach | `agent-reach reddit` |
| YouTube | Agent-Reach | `agent-reach youtube` |
| 小红书 | Agent-Reach | `agent-reach xiaohongshu` |
| 通用网页 | chrome-cdp | `cdp.mjs html` |
| 搜索 | web_search | OpenClaw 内置 |

## 采集规范

- 遵守平台 robots.txt
- 控制采集频率，避免被封
- 敏感数据加密存储
- 定期清理过期数据
