# B 站评论图片报告 - 部署指南

## 🌐 方案 1：GitHub Pages（推荐 ⭐）

### 优势
- ✅ 完全免费
- ✅ 永久链接
- ✅ 支持自定义域名
- ✅ 自动 HTTPS
- ✅ 访问速度快

### 部署步骤

#### 1️⃣ 创建 GitHub 仓库
```bash
# 在 GitHub 创建新仓库，例如：bili-comments-report
# 或者用现有仓库
```

#### 2️⃣ 上传文件
```bash
cd /Users/wh1ko/Documents/openclaw/agents/hunter/bili_images/

# 初始化 git（如果还没有）
git init

# 添加所有文件
git add .

# 提交
git commit -m "B 站评论图片报告"

# 关联远程仓库（替换为你的仓库地址）
git remote add origin https://github.com/你的用户名/bili-comments-report.git

# 推送
git push -u origin main
```

#### 3️⃣ 启用 GitHub Pages
1. 打开仓库页面
2. 点击 **Settings** → **Pages**
3. Source 选择 `main` 分支 → **Save**
4. 等待 1-2 分钟

#### 4️⃣ 获取在线链接
```
https://你的用户名.github.io/bili-comments-report/B 站评论图片报告.html
```

## 🌐 方案 2：Vercel（更简单 ⭐⭐）

### 优势
- ✅ 无需配置
- ✅ 自动部署
- ✅ 全球 CDN
- ✅ 免费 HTTPS

### 部署步骤

#### 1️⃣ 安装 Vercel CLI
```bash
npm install -g vercel
```

#### 2️⃣ 部署
```bash
cd /Users/wh1ko/Documents/openclaw/agents/hunter/bili_images/
vercel --prod
```

#### 3️⃣ 获取链接
```
https://xxx.vercel.app/B 站评论图片报告.html
```

## 🌐 方案 3：Netlify Drop（最简单 ⭐⭐⭐）

### 优势
- ✅ 拖拽上传

- ✅ 30 秒上线

2. 把 `bili_images` 文件夹拖到网页

## 🌐 方案 4：阿里云 OSS（国内访问快）

### 优势
- ✅ 国内访问速度快
- ✅ 稳定可靠
- ✅ 费用低（约 0.1 元/天）

### 部署步骤

#### 1️⃣ 创建 OSS Bucket
1. 登录阿里云控制台
2. 创建 OSS Bucket（选择公共读）
3. 获取 Endpoint

#### 2️⃣ 上传文件
```bash
# 使用 ossutil 工具
ossutil cp -r /Users/wh1ko/Documents/openclaw/agents/hunter/bili_images/ oss://你的 bucket/
```

#### 3️⃣ 获取链接
```
https://你的 bucket.oss-cn-hangzhou.aliyuncs.com/B 站评论图片报告.html
```
