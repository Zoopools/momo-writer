## 📖 学习重点

### 1. 设计哲学（核心）

**5 种风格定位**:
| 风格 | 适用场景 | 核心特征 |
|------|---------|---------|
| **Future & Depth** | 前沿科技、产品发布 | 毛玻璃 + 深色模式、光晕和透明感 |
| **Efficiency & Speed** | 专业工具、数据看板 | 干净扁平 + Bento UI、清晰边界、模块化 |
| **Trust & Professional** | 金融、正式演示、企业方案 | 瑞士极简、大量留白、依赖排版和严格网格 |
| **Care & Resonance** | 人文、生活、品牌故事 | 低饱和度自然色 + 超大圆角、超柔漫射阴影 |
| **Immersion & Expression** | 娱乐、叙事、创意展示 | 拟物材质 + 高对比情绪色、打破常规网格 |

### 2. 空间与排版

**密度层级**: 密度与重要性成反比
- 核心焦点：低密度/大边距
- 数据列表：高密度/小边距

**排版系统**:
- 字体：现代无衬线字体（Clash Display, Satoshi, DM Sans）
- 标题与正文要有明显的 weight 和 size 对比
- 正文行高：`leading-[1.5]` 或 `leading-[1.6]`
- 最小可读尺寸：`12px`（注释）/`14px/16px`（正文）

### 3. 系统完整性约束

**所有设计决策必须映射到有限变量集**：

| 系统 | 约束 |
|------|------|
| **色彩系统** | 主色定义品牌；互补色强引导；类似色柔引导 |
| **空间间距（8 点网格）** | 间距限定：`8/12/16/20/24/32/40` |
| **圆角** | 默认 `rounded-[12px]`，Care 风格可用 `rounded-[24px]` 或 `rounded-full` |
| **尺寸最小值** | 最小点击热区 `44px`；最小可读 `12px` |
| **阴影控制** | 必须使用柔光如 `shadow-[0_10px_30px_rgba(0,0,0,0.08)]` |

### 4. 2026 新兴设计趋势

**趋势 1: Ghostly Agency（幽灵代理）**
- 预测性存在（渐进式内容揭示）
- 隐形管家（半透明元素 + 微妙浮动动画）
- 意图可视化（光晕/粒子暗示"思考"）

**趋势 2: 粗砺真实（The Grain of Truth）**
- 数字褶皱（SVG 噪点纹理）
- 有机排版（轻微旋转、非严格对齐）
- 触觉质感（颗粒感背景 + 不规则边框）

**趋势 3: Soft Pop & Playful Geometry**
- 糖果色 + 几何图形
-  playful 动效

**趋势 4: Neo-Brutalism / Anti-Design**
- 高对比、粗边框、反传统
