## 视频制作全流程

```
选题策划 → 脚本撰写 → 素材录制 → 后期剪辑 → 封面标题 → 发布推广 → 数据复盘
   ↓          ↓          ↓          ↓          ↓          ↓          ↓
 1-2 天     1-2 天      1 天       2-3 天      半天       半天       持续
```

**总周期**: 7-10 天/条视频（精品）  
**快速生产**: 3-5 天/条视频（常规）

## 前期准备

### 1. 选题策划

#### 选题来源

| 来源 | 说明 | 频率 |
|------|------|------|
| **热点追踪** | 新游戏发布、版本更新、电竞赛事 | 每周 |
| **用户需求** | 评论区提问、社群讨论 | 持续 |
| **竞品分析** | 同类 UP 主爆款选题 | 每周 |
| **系列规划** | 固定栏目、连载内容 | 每月 |

#### 选题评估标准

| 维度 | 权重 | 评估要点 |
|------|------|---------|
| **热度** | 30% | 搜索量、讨论度、时效性 |
| **差异化** | 25% | 独特视角、独家内容 |
| **可执行性** | 25% | 素材获取、制作难度 |
| **变现潜力** | 20% | 商单匹配、带货可能 |

#### 选题模板

```markdown
# 选题卡片

**标题**: (暂定)
**核心看点**: (一句话概括)
**目标受众**: (谁会用/会看)
**对标视频**: (参考链接)
**预计时长**: (分钟)
**制作周期**: (天)
**预期数据**: (播放/点赞/评论)
```

---

### 2. 脚本撰写

#### 脚本结构

```
开头 (0-30 秒) → 引入 (30-60 秒) → 主体 (1-5 分钟) → 高潮 (5-7 分钟) → 结尾 (7-8 分钟)
    ↓              ↓              ↓              ↓             ↓
 钩子引入      背景介绍       核心内容       亮点展示      总结 + 引导
```

#### 开头钩子 (前 5 秒定生死)

**有效钩子类型**:

| 类型 | 示例 | 适用场景 |
|------|------|---------|
| **痛点型** | "你是不是也遇到过..." | 教程类 |
| **悬念型** | "没想到竟然是这样..." | 测评类 |
| **对比型** | "之前 vs 现在，差距太大了" | 评测类 |
| **数字型** | "3 个技巧帮你..." | 干货类 |
| **冲突型** | "90% 的人都做错了" | 纠错类 |

#### 脚本模板

```markdown
# 视频脚本

## 开头 (0-30 秒)
[钩子]: (吸引注意力的话术)
[画面]: (对应画面描述)

## 引入 (30-60 秒)
[背景]: (为什么要做这个视频)
[预告]: (视频会讲什么)

## 主体 (1-5 分钟)
[要点 1]: (第一个核心点)
  - 画面:
  - 台词:
[要点 2]: (第二个核心点)
  - 画面:
  - 台词:
[要点 3]: (第三个核心点)
  - 画面:
  - 台词:

## 高潮 (5-7 分钟)
[亮点]: (最精彩的部分)
[反转]: (意外发现/结果)

## 结尾 (7-8 分钟)
[总结]: (回顾要点)
[引导]: (点赞/评论/关注)
[预告]: (下期内容)
```

## 录制阶段

### 1. 游戏录制

#### 录制软件

| 软件 | 平台 | 优点 | 缺点 |
|------|------|------|------|
| **OBS** | 全平台 | 免费、功能强大 | 需要配置 |
| **NVIDIA ShadowPlay** | Windows | 性能好、方便 | 需要 N 卡 |
| **AMD ReLive** | Windows | 性能好 | 需要 A 卡 |
| **Mac 自带录制** | macOS | 简单 | 功能少 |

#### 录制参数建议

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| **分辨率** | 1920x1080 | 1080p 主流 |
| **帧率** | 60fps | 游戏视频标配 |
| **码率** | 10000-20000 kbps | 保证画质 |
| **格式** | MP4 (H.264) | 兼容性好 |
| **音频** | 48kHz, AAC | 音质清晰 |

#### 录制注意事项

- ✅ 关闭游戏内 HUD（需要时保留）
- ✅ 清理桌面通知
- ✅ 检查麦克风音量
- ✅ 预留片头片尾素材
- ✅ 录制多份备用

---

### 2. 配音录制

#### 配音环境

| 要求 | 说明 |
|------|------|
| **安静** | 关闭空调、风扇等噪音源 |
| **吸音** | 用被子/窗帘减少回声 |
| **设备** | USB 麦克风即可（如 Blue Yeti） |
| **软件** | Audacity（免费）或 Adobe Audition |

#### 配音技巧

- ✅ 保持微笑（声音会更有感染力）
- ✅ 语速适中（比平时稍快）
- ✅ 断句清晰（方便后期剪辑）
- ✅ 多录几遍（选择最好的）

## 后期制作

### 1. 剪辑软件

| 软件 | 难度 | 价格 | 适用 |
|------|------|------|------|
| **剪映** | ⭐ | 免费 | 新手、快速出片 |
| **Premiere Pro** | ⭐⭐⭐ | 订阅 | 专业制作 |
| **Final Cut Pro** | ⭐⭐ | 买断 | Mac 用户 |
| **DaVinci Resolve** | ⭐⭐⭐ | 免费 | 调色专业 |

### 2. 剪辑流程

```
导入素材 → 粗剪 → 精剪 → 配音 → 字幕 → 特效 → 调色 → 导出
   ↓         ↓       ↓       ↓       ↓       ↓       ↓       ↓
 整理素材   去废片   节奏调整  对齐画面  添加字幕  转场特效  色彩统一  平台适配
```

### 3. 剪辑要点

#### 节奏控制

| 视频类型 | 平均镜头时长 | 说明 |
|---------|------------|------|
| **快节奏** | 2-3 秒 | 游戏集锦、高燃混剪 |
| **中节奏** | 3-5 秒 | 教程、测评 |
| **慢节奏** | 5-8 秒 | 剧情、深度分析 |

#### 转场使用

- ✅ 硬切（最常用，不突兀）
- ✅ 淡入淡出（场景转换）
- ⚠️ 特效转场（慎用，容易土）

#### 字幕规范

| 要素 | 要求 |
|------|------|
| **字体** | 思源黑体/站酷高端黑 |
| **大小** | 40-60px（1080p） |
| **颜色** | 白色 + 黑色描边 |
| **位置** | 底部 1/3 处 |
| **时长** | 与配音同步 |

## 发布优化

### 1. 标题优化

#### 标题公式

```
【类型标签】+ 核心看点 + 情绪词/数字 + 目标受众

示例:
【原神攻略】3 个技巧帮你轻松满星深渊，手残党也能学会！
```

#### 标题检查清单

- [ ] 包含核心关键词
- [ ] 有数字/对比/悬念
- [ ] 明确目标受众
- [ ] 长度适中（20-30 字）
- [ ] 无违规词

### 2. 封面设计

#### 封面要素

| 要素 | 要求 |
|------|------|
| **主体** | 清晰、突出（角色/场景） |
| **文字** | 少而精（3-5 个字） |
| **色彩** | 鲜艳、对比强 |
| **情绪** | 夸张、有感染力 |

#### 封面尺寸

| 平台 | 尺寸 | 比例 |
|------|------|------|
| **B 站** | 1920x1080 | 16:9 |
| **抖音** | 1080x1920 | 9:16 |
| **视频号** | 1080x1080 | 1:1 |

### 3. 标签选择

| 类型 | 数量 | 说明 |
|------|------|------|
| **核心标签** | 2-3 个 | 视频主题关键词 |
| **长尾标签** | 2-3 个 | 细分领域词 |
| **热点标签** | 1-2 个 | 当前热门话题 |

## 数据复盘

### 1. 核心指标

| 指标 | 含义 | 优化方向 |
|------|------|---------|
| **播放量** | 曝光程度 | 标题、封面、选题 |
| **完播率** | 内容吸引力 | 开头、节奏、内容质量 |
| **互动率** | 粉丝粘性 | 引导评论、内容共鸣 |
| **转粉率** | 转化能力 | 人设、内容价值 |

### 3. 优化迭代

| 问题 | 可能原因 | 优化方案 |
|------|---------|---------|
| 播放低 | 标题/封面不行 | A/B 测试、参考爆款 |
| 完播低 | 开头不吸引人 | 优化前 5 秒、加快节奏 |
| 互动低 | 缺少引导 | 增加提问、设置话题 |
| 转粉低 | 人设不清晰 | 强化个人特色、固定栏目 |
